ZZZ_MasterPatch_Lore_And_Fix v1.0 — CK3 1.17.0

Install: place this folder into:
  Documents\Paradox Interactive\Crusader Kings III\mod\

Run:
 - Run make_blank_dds.py once (pip install pillow) to create placeholder icons.
 - Enable mod in Irony/Launcher. Place this mod ABOVE cosmetic portrait/textures.
 - Start a fresh game (recommended) and test.

Features:
 - Aerulean faith tweaks (cousin-only marriage, tenet changes)
 - Hidden Elf start, Age progression (manual), event arcs: Sundering → Exile → Gathering → Reckoning → Dominion → Harmony
 - Midnight Covenant (secret order): initiation, protection, high intrigue/seduction
 - Grand Matriarch court position; matriarch betrayal & resolution events
 - Ceremonies, festivals, petitions, incognito visits, medals, sibling-marriage enforcement
 - Sandbox enabled (all tenets visible; max 30 tenets). Remove common/script_values/zzz_sandbox_values.txt to disable.
